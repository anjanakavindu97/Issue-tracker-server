# Issue Tracker / Nodejs + TypeScript

Clone the repository

Change the directory to clone the repository

## Install the npm packages

```js
npm i
```

## Set the environment variable

Create a new file name ``.env`` inside the cloned project folder. Copy and paste the code

```js
PORT=3000
NODE_ENV=development
MONGOOSE_URI=mongodb://0.0.0.0:27017/issue-tracker
JWT_SECRET=
```

## Run the project

```js
nodemon
```
